### Ollama server
* Install Ollama (choose the relevant OS)
```
curl -fsSL https://ollama.com/install.sh | sh
```
* If you are under a proxy:
- set HTTP_PROXY
- set HTTPS_PROXY
* To run Ollama
```
ollama serve
```
* To download a model
```
ollama pull MODEL
```
* To see the list of downloaded models
```
ollama list
```
